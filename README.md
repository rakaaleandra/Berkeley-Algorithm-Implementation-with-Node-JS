# Berkeley-Algorithm-Implementation-with-Node-JS
Menggunakan 3 Terminal 

Jalankan Server Backend ( Server )

```javascript
node server.js
```

Jalankan Server Web ( Front End )
```javascript
npx http-server -p 9000
```

Menjalankan client pakai

```javascript
node client.js [client_id]
```
Contoh : 

```javascript
node client.js 39
```

Mengakses Dashboard
- Dashboard Server Utama : http://localhost:9000/server.html
- Dashboard Client Individual : http://localhost:9000/client.html?id=1 , ganti id sesuai dengan client_id
